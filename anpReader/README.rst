anp_reader
=============

Brazilian ANP gas station and price processor.

To install it
-------------

::

    $ sudo pip install anp_reader

Usage
-----

::

    usage: anp_reader [-h]

    anp_reader

    optional arguments:
      -h, --help            show this help message and exit
